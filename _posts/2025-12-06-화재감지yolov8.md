개요
-----------------
본 글은 2024년도에 진행한 캡스톤 프로젝트 "투척용 소화구를 발사하는 AI 소방 RC카"를 제작하기 위하여 YOLOv8로 화재 이미지를 학습 시키는 과정에 대하여 작성한 글입니다.

S/W 개발 환경
-----------------
* Desktop
  * CUDA
    * NVIDIA의 General-Purpose computing on Graphics Processing Units(GPGPU)를 활용할 수 있게 해 주는 소프트웨어로 CPU가 맡았던 연산을 GPU에서 사용해 연산 속도를 향상시키며, CUDA 코어가 장착된 NVIDIA GPU에서 작동한다. 본 프로젝트에서는 NVIDIA GeForce RTX 2080 SUPER와 호환되는 CUDA 11.8버전을 사용하여, AI 모델 학습과 학습된 결과를 통한 객체 인식을 가능하게 하였다.
  * cuDNN(cuda Deep Neural Network Library)
    * 딥러닝 및 인공 신경망 프레임워크를 가속화 하기 위한 라이브러리로 GPU 가속화 라이브러리의 기초 요소이며, 컨볼루션(Convolution), 풀링(Pooling), 표준화(Normarization), 활성화(Activation)과 같은 것들을 빠르게 실행할 수 있게 한다. 본 프로젝트에서는 CUDA 11.8버전과 호환되는 cuDNN 8.9.1버전을 사용하였다.
  * Visual studio code
    * Microsoft에서 개발한 텍스트 에디터로, 본 프로젝트에서 코드 작성을 위해 사용하였다.
  * Python
    * 인터프리터 프로그래밍 언어로, 직관적이고 쉬운 문법과 다양한 라이브러리를 제공한다. 본 프로젝트에서는 CUDA 11.8버전과 호환되는 Python 3.8.10버전을 사용하여 가상환경을 구축하였다. 가상환경에 추가로 설치한 라이브러리는 tensorflow 2.10, tensorflow-gpu 2.10, ultralytics 8.2, pytorch(torch 2.3.1+cuda118, torchvision 0.18.1+cuda118, torchaudio 2.3.1+cuda118), numpy 1.23.5, OpenCV 4.10가 있다.


* Raspberry Pi 4B
  * Python
    * Desktop과 동일하게 Python 3.8.10버전을 사용하였으며, OpenCV를 추가적으로 사용하고, RC카 제어를 위해 Yahboom에서 제공하는 소스 코드들을 수정하여 사용하였다.
   

* 활용 source
  * AI-Hub
    *  AI 기술 및 제품·서비스 개발에 필요한 AI 인프라(AI 데이터, AI SW API, 컴퓨팅 자원)를 지원함으로써 누구나 활용하고 참여하는 AI 통합 플랫폼이다. 화재감지 AI 모델 학습을 위한 화재 관련 이미지 160만여 장을 내려받았다.
